## 1. 프로젝트 개요

### 이 프로젝트가 무엇인가

**유성선병원 성형외과 신설** — 개인 의원 개원이 아니라 **기존 종합병원(유성선병원) 내 신규 진료과 신설**이라는 점이 핵심입니다.

이 구분이 브랜딩 전략 전체의 출발점입니다.

### 성공의 조건

> "미용을 앞세우지 않고, **종합병원급 안전 시스템 + 성형외과의 섬세함**을 한 문장으로 묶어 **신뢰 기반 전환**을 만든다"

개인 성형외과와의 차별화는 **가격이나 기술이 아닌 안전 인프라**입니다. 이 병원에서만 가능한 차별점(24시간 응급, 마취과 상주, 다과 협진)을 전면에 내세웁니다.

### 운영 구조 — 2트랙

| 트랙 | 채널 | 목적 |
|------|------|------|
| **Track A** (본원 채널) | 유성선병원 홈페이지 · 대표 SNS | 신뢰 · 공식성 · 인지 확산 |
| **Track B** (전용 채널) | 성형외과 전용 마이크로사이트 + 인스타 + 네이버 플레이스 | 정보 · 교육 · 상담 전환 |

Track A가 신뢰를 만들고, Track B가 전환을 만드는 구조입니다.

## 2. 브랜드 포지셔닝

### 핵심 한 문장

> **"종합병원급 안전성과 성형외과의 섬세함을 함께 제공한다."**

### 환자 언어 원칙 (의료 용어 → 친숙한 표현)

실무에서 콘텐츠 제작 시 반드시 적용해야 하는 번역 기준입니다.

| 의료 용어 (사용 금지) | 환자 언어 (사용) |
|----------------------|----------------|
| 상안검 / 하안검 | 처진 눈꺼풀 / 눈밑 지방 |
| HIFU / 써마젯 등 장비명 난사 | 얼굴 처짐 해결 / 탄력 레이저 (울쎄라·써마지) |
| 미세봉합 / 열상 | 상처 봉합 / 흉터 최소화 |

### 의료진 브랜딩 원칙

신정환 과장의 커뮤니케이션은 **전문성·리더십·공신력** 중심으로 제한합니다.

**절대 금지**
- 과장 · 비교 · 단정 표현

## 3. 5대 진료축 전략

대시보드의 **5대 센터** 섹션에 해당합니다. 각 진료 분야는 전략적 역할이 다릅니다.

### 진료축별 전략 역할

| # | 진료축 | 대표 키워드 | 전략적 역할 | 컬러 |
|---|--------|------------|------------|------|
| 1 | **쁘띠/스킨** | 보톡스, 레이저, 필러 | 반복 내원 · 유입 확대 (입구 역할) | 블루 |
| 2 | **리프팅** | 울쎄라, 써마지 | 고수익 · 프리미엄 신뢰 | 퍼플 |
| 3 | **눈** | 쌍꺼풀, 하안검 | 시그니처 전환축 (핵심 상담 유도) | 앰버 |
| 4 | **외상** | 안면미세봉합, 안면골절 수술 | 종합병원만의 독보 차별화 | 레드 |
| 5 | **재건/종괴** | 몽우리, 종괴 | 의료 신뢰의 정점 · 불안 해소 | 에메랄드 |

### 구조적 포인트

- **쁘띠/스킨**은 진입 장벽이 낮아 첫 내원을 유도하는 입구 역할
- **외상·재건**은 타 성형외과에 없는 차별화 콘텐츠 — SNS에서 독보적 소재로 활용
- **눈**이 실질적 전환의 핵심축 — 상담 후 수술 결정율이 가장 높음

### 홈페이지 카테고리 구조 (2카테고리 스플릿)

기존 "5대 센터 나열"에서 **2카테고리 스플릿 배너 방식**으로 전환했습니다.

```
┌─────────────────────┬─────────────────────┐
│     미용·성형        │      치료·재건       │
│  (테라코타 레드)     │   (다크 네이비)      │
│                     │                     │
│  · Eye Center       │  · Trauma           │
│  · Lifting          │  · Reconstruction   │
│  · Petit/Skin       │                     │
│                     │                     │
│ "아름다움을 섬세하게" │ "종합병원의 안전함으로"│
└─────────────────────┴─────────────────────┘
    ↕ 호버 시 확장 (1.6배) + CTA 버튼 출현
```


## 7. 상담 마케팅 전략

대시보드의 **상담 마케팅 전략** 섹션에 해당합니다.

### 채널 1 — 카카오톡 케어챗

본원에서 이미 운영 중인 시스템을 **성형외과 전용으로 확장**. 추가 비용 최소화.

**핵심 기능**

| 기능 | 내용 |
|------|------|
| 24시간 대응 | 자동 응답 + 담당자 연결 |
| 예약 연결 | 카카오톡 → 예약 시스템 직접 연동 |
| 이력 관리 | 문의 내용 · 관심 시술 · 상담 단계 데이터화 |
| 메시지 발송 | 예약 확인 · 사후 관리 · 이벤트 안내 |

**활용 시나리오**

```
① 초기 문의:
   "쌍꺼풀 상담 가능한가요?"
   → 진료 가능 시간 + 예약 링크 안내

② 시술 후 관리:
   시술 후 1일/3일/7일 자동 메시지로 회복 상태 확인

③ 재방문 유도:
   관심 표현 고객에게 신규 시술/이벤트 정보 제공
```

### 채널 2 — 인스타그램 DM

전용 인스타그램 계정에서 **DM 상담 전담 체계** 구축.

| 항목 | 기준 |
|------|------|
| 응답 시간 | 영업시간 내 2시간 이내, 이후 익일 오전 우선 |
| 댓글 관리 | 24시간 이내 답변 → DM 유도로 전환 |
| 금지 사항 | 가격 단정 금지 / 효과 보장 금지 / 타 병원 비교 금지 |

**주요 응답 템플릿**

```
"안녕하세요, 유성선병원 성형외과입니다.
 문의 주신 [시술명]에 대해 안내해 드릴게요."

"정확한 상담은 직접 뵙고 진행되지만,
 궁금하신 부분 간단히 안내해 드릴게요."

"상담 예약은 이 링크로 가능합니다. [예약 링크]"
```

### 통합 상담 운영 원칙

| 항목 | 기준 |
|------|------|
| 카카오톡 응답 | **30분 이내** 1차 응답 |
| 인스타 DM 응답 | **2시간 이내** (영업시간 기준) |
| 상담 범위 | 시술 설명 · 예약 안내 · 회복 기간 (가격은 범위만) |
| 전환 목표 | 온라인 상담 → 전화 → 내원 순차 유도 |
| 담당 인력 | 초기 1명 전담 → 문의량 증가 시 2명 체제 |

**기대 효과**

- 문의 → 예약 전환율 **15~20%** 목표
- 24시간 대응으로 잠재 고객 이탈 방지
- 상담 데이터 축적 → 맞춤형 콘텐츠 제작 피드백 루프

## 8. 12주 콘텐츠 캘린더

대시보드의 **12주 콘텐츠 캘린더** 섹션에 해당합니다.

**원칙**: 전후 사진 없이 "교육 / 과정 / 공감" 중심으로 설계.
**규모**: 주 3편 릴스 × 12주 = **총 36개 콘텐츠 제목 확정**

### 전체 캘린더 (주차별 요약)

| 주차 | 테마 | 대표 제목 예시 |
|------|------|--------------|
| **Week 1** | 오픈 인지 | 성형외과 신설, 뭐가 달라지나요? |
| **Week 2** | 눈 1: 기능/인상 | 눈이 피곤해 보이는 진짜 이유 |
| **Week 3** | 리프팅 1: 원리/선택 | 울쎄라 vs 써마지, 뭐가 다를까요? |
| **Week 4** | 쁘띠 1: 안전 | 보톡스가 '안 듣는' 것처럼 느껴질 때 |
| **Week 5** | 스킨/레이저 1 | 레이저는 '횟수'보다 '간격'이 더 중요합니다 |
| **Week 6** | 외상 1 | 미세봉합, '빨리'보다 중요한 한 가지 |
| **Week 7** | 눈 2: 회복 | 눈 수술 후 '붓기'에 대한 현실적인 설명 |
| **Week 8** | 리프팅 2: 연령/설계 | 30대 리프팅, '지금' 하면 좋은 이유/아닌 이유 |
| **Week 9** | 재건/종괴 | 얼굴 '몽우리'가 생겼을 때 제일 먼저 할 것 |
| **Week 10** | 쁘띠/스킨 2 | 행사/면접 전, 시술 타이밍 어떻게 잡나요? |
| **Week 11** | 신뢰 강화 | 저희가 상담에서 '안 한다'고 말하는 경우 |
| **Week 12** | 전환/안정 | 처음 오시는 분: 진료축별 예약 가이드 |

### 콘텐츠 흐름의 설계 논리

```
Week 1     → 인지: "어떤 병원인가?"
Week 2~5   → 교육: 각 진료축별 기초 정보
Week 6     → 차별화: 외상 (타 계정에 없는 소재)
Week 7~10  → 심화: 각 진료축 2라운드 (회복·선택·설계)
Week 11    → 신뢰: 솔직함으로 역설적 신뢰 구축
Week 12    → 전환: 예약 유도 + 다음 시즌 씨드
```

## 9. 오픈 이벤트 & 브랜드 조사

대시보드의 **이벤트** 섹션에 해당합니다.

### 오픈 기념 이벤트

- 첫 상담 내원 고객 대상 소정의 혜택 제공 (내용 추후 확정)

- 원내 X배너·안내 리플렛과 연계해 오프라인 노출

### 유성선병원 60주년 기념 연계

> "60년 신뢰의 병원 + 새로운 성형외과" — 안전성 메시지 강화

- 병원 전체 60주년 행사에 성형외과 신설을 함께 알리는 기회로 활용
- 본원 Track A 채널과 연계해 기관 공신력 확산

### 브랜드 인식 조사

**목적 & 방법**

- 내원 고객 및 온라인 설문으로 초기 인지도·이미지 데이터 수집
- 타 성형외과 병원 대비 유성선병원 성형외과의 인식 차이 파악
- 선택 이유, 신뢰 요소, 개선 요구사항 등 정성·정량 데이터 확보

**주요 조사 항목**

| 항목 | 내용 |
|------|------|
| 브랜드 인지도 | 최초 인지 / 보조 인지 |
| 신뢰 요인 | 선택 근거 상위 요소 |
| 경쟁 비교 | 타 성형외과 대비 인식 |
| 개선 요구 | 불안 요소 / 기대 요소 |

**활용 방향**: 수집된 데이터 → 콘텐츠 메시지 고도화 · 광고 타겟 조정 · 실질 마케팅 전략 재반영

## 10. 실행 타임테이블

대시보드의 **일정** 섹션에 해당합니다.

### 전체 타임라인 (11주 준비)

```
D-11W ~ D-8W  ────  기반 세팅
D-7W  ~ D-4W  ────  콘텐츠 자산 제작
D-3W  ~ 오픈  ────  실행 준비 완성
오픈 이후     ────  광고 + 콘텐츠 루틴 운영
```

### 단계별 상세

#### Phase 1 — 기반 세팅 (D-11W ~ D-8W, 약 3주)

| 항목 | 기간 | 예산 |
|------|------|------|
| 공식 채널 개설 / 스킨 통일 | 2주 | — |
| 전용 인스타·유튜브·플레이스·카카오 문의 동선 확정 | 2주 | — |
| 홍보 인쇄물 제작 (X배너, 리플렛, 원내 게시물) | 2주 | 100만 이내 |

#### Phase 2 — 콘텐츠 자산 제작 (D-7W ~ D-4W, 약 3주)

| 항목 | 기간 | 예산 |
|------|------|------|
| 자체 모델/콘텐츠 제작 (영상+스틸컷, '과정/교육' 중심) | 2주 | 500만 |
| 홈페이지 최종 확정 및 배포 | — | 800~1,000만 |

> 홈페이지 6페이지 초안은 이미 완료. 실제 이미지·의료진 사진·최종 카피 반영 후 확정.

#### Phase 3 — 실행 준비 완성 (D-3W ~ 오픈)

| 항목 | 내용 |
|------|------|
| X배너 등 원내 게시물 | 1층/입구 핵심 동선 설치 |
| 네이버 플레이스 1차 완성 | 사진 10장 + FAQ 20개 + 소식 1개 업로드 |
| 릴스 선제작 | **1~4주 분량 (12편)** 오픈 전 완성 → 오픈 직후 루틴 즉시 가동 |

#### Phase 4 — 광고 운영 (오픈 후)

> 초기 8주: **300~500만/월 테스트** → 데이터 확인 후 증액 결정
>
> (사용자 메모의 "월 광고 예산 2,000만" 대비 초기 과도한 지출 방지)
