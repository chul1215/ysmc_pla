# 유성선병원 성형외과 신설 홍보 마케팅 기획안
## 경쟁분석 포함 종합 보고서

> **작성 기준일**: 2026년 3월 | **기획 단계**: 실행 준비 완료 (2분기 오픈 예정)

## 1. 프로젝트 개요 및 핵심 지표

### 1.1 병원 프로필

| 항목 | 내용 |
|------|------|
| **병원명** | 유성선병원 성형외과 |
| **성격** | 종합병원 내 신설 전문과 |
| **담당의** | 신정환 과장 (가톨릭중앙의료원 출신) |
| **위치** | 대전 유성구 |
| **핵심 메시지** | "대학병원의 안전함에 섬세함을 더하다" |
| **진료 개시** | 2026년 2분기 |

### 1.2 핵심 기획 지표

| 지표 | 수치 |
|------|------|
| 5대 진료축 | 5개 센터 |
| 홈페이지 구성 | 6페이지 (신규) |

| 실행 준비 기간 | 11주 (D-11W) |

### 1.3 이 프로젝트의 핵심

**유성선병원 성형외과 신설** — 개인 의원 개원이 아니라 **기존 종합병원(유성선병원) 내 신규 진료과 신설**이라는 점이 브랜딩 전략 전체의 출발점입니다.

> "미용을 앞세우지 않고, **종합병원급 안전 시스템 + 성형외과의 섬세함**을 한 문장으로 묶어 **신뢰 기반 전환**을 만든다"

개인 성형외과와의 차별화는 **가격이나 기술이 아닌 안전 인프라**입니다. 이 병원에서만 가능한 차별점(24시간 응급, 마취과 상주, 다과 협진)을 전면에 내세웁니다.

### 1.4 운영 구조 — 2트랙

| 트랙 | 채널 | 목적 |
|------|------|------|
| **Track A** (본원 채널) | 유성선병원 홈페이지 · 대표 SNS | 신뢰 · 공식성 · 인지 확산 |
| **Track B** (전용 채널) | 성형외과 전용 마이크로사이트 + 인스타 + 네이버 플레이스 | 정보 · 교육 · 상담 전환 |

Track A가 신뢰를 만들고, Track B가 전환을 만드는 구조입니다.

> **"유성구 유일 — 종합병원의 안전 인프라 위에서, 성형외과 전문의가 직접 집도하는 미용과 재건"**

### 2.3 포지셔닝 4대 축

| 포지셔닝 요소 | 경쟁 우위 내용 |
|--------------|--------------|
| **유성구 입지 독점** | 유성구 내 미용 성형외과 개원의원 부재. 반경 내 직접 경쟁 없음 |
| **종합병원 인프라** | 마취과 전문의 상주, 중환자실, 응급 체계 — 개원의원이 가질 수 없는 안전 우위 |

## 3. 5대 진료축 전략

| # | 진료축 | 대표 시술 | 전략적 역할 | 컬러 |
|---|--------|----------|------------|------|
| 1 | **쁘띠/스킨** | 보톡스, 레이저, 필러 | 반복 내원 / 유입 확대 (Entry 역할) | 블루 |
| 2 | **리프팅** | 울쎄라, 써마지 | 고수익 / 프리미엄 신뢰 형성 | 퍼플 |
| 3 | **눈** | 쌍꺼풀, 하안검 | 시그니처 전환축 (핵심 상담 유도) | 앰버 |
| 4 | **외상** | 안면미세봉합, 안면골절 수술 | 독보적 차별화 | 레드 |
| 5 | **재건/종괴** | 몽우리, 종괴 | 의료 신뢰 정점 / 불안 해소 | 에메랄드 |

> **전략 해설**: 쁘띠/스킨으로 반복 접점을 만들고 → 눈/리프팅으로 고가 전환 → 외상/재건으로 종합병원 신뢰를 독점하는 구조.

**구조적 포인트**
- **쁘띠/스킨**은 진입 장벽이 낮아 첫 내원을 유도하는 입구 역할
- **외상·재건**은 타 성형외과에 없는 차별화 콘텐츠 — SNS에서 독보적 소재로 활용
- **눈**이 실질적 전환의 핵심축 — 상담 후 수술 결정율이 가장 높음

## 4. SWOT 분석

### 강점 (Strengths)

- 종합병원 안전 인프라 완비 (마취과, 중환자실, 응급체계)
- 가톨릭중앙의료원 출신 전문의의 학술적 배경
- 미용(눈, 리프팅, 쁘띠)부터 치료(외상, 재건, 종괴)까지 풀커버
- 유성구 내 성형외과 전문 의원 부재 → 지역 독점 가능
- 기존 유성선병원 환자 베이스 (타과 연계 진료)

### 약점 (Weaknesses)

- 신설 부서 인지도 제로
- "미용 성형은 전문 의원에서"라는 소비자 인식
- 성형외과 수요 집중 상권(둔산동)과 물리적 거리
- 종합병원 특성상 공격적 할인/이벤트 마케팅 어려움
- 1인 전문의 체제 — 수술 볼륨·대기시간 한계

### 기회 (Opportunities)

- 유성구 성형외과 수요 공백 → 둔산동까지 이동하던 환자 흡수
- 성형 부작용 이슈 증가로 "안전한 수술 환경" 수요 상승
- 외상/재건 니치 — 오체안을 제외하면 전문 의원 부재
- 타과 협진 가능 (안과-눈, 이비인후과-코, 피부과-스킨)
- "유성구 성형외과", "대전 외상 성형" 등 비경쟁 네이버 SEO 키워드 선점
- 환자 3~4개 병원 리뷰 비교 행동 → 신뢰 콘텐츠로 전환 유도

### 위협 (Threats)

- 둔산동 성형외과 8개+ 반경 10분 내 밀집
- 서울 강남 KTX 1시간대 — 고가 수술 환자 유출
- 닥터스미 3인 전문의 체제로 분야별 전문성 어필
- 을지대병원 — "종합병원 성형외과" 포지션 기존 점유
- 건양대병원 상급종합병원 타이틀 — 안전성 신뢰도 최고 수준
- 오체안의 외상/재건 10,000건 이상 수술 실적
- 의료광고 규제 — 종합병원의 적극적 미용 마케팅 제약
- 이벤트 가격 경쟁 압박

### 주차별 상세 콘텐츠 (36개 제목)

#### Week 1 — 신설 부서 오픈

- 채널 개설 + 상담 기능 활성화
- "1:1 전문의 직접 상담" 메시지로 의원 대비 차별화
- 정기 건강정보 메시지로 리텐션 강화

#### C. 웹사이트 (현재 프로젝트)

- 5개 진료축별 상세 페이지 완비 → 네이버 SEO 랜딩 최적화
- CTA 3종 배치 (상담예약/전화/카카오톡) 완료
- 추가 제안: "종합병원에서 성형수술을 받는 이유" 등 안전 관련 콘텐츠 보강

#### D. 전문 플랫폼 (3순위, 미용 시술 활성화 시)

- 강남언니, 바비톡, 모두닥 입점 검토 (쁘띠/스킨 시술 리뷰 우선 축적)

#### E. 유튜브/인스타그램 (중장기)

- 신정환 과장 전문가 브랜딩 영상 (수술 철학, 의료 지식, Q&A)
- 종합병원 시설 투어 영상 (수술실, 입원실, 마취 과정 등 안전 인프라 시각화)

### 14.3 경쟁 대응 전략 요약

#### vs. 둔산동 개원의원

| 경쟁 이슈 | 대응 전략 |
|-----------|-----------|
| "의원이 더 전문적" 인식 | 전담 전문의 체제 강조, 학력/경력 상세 공개 |
| 가격 경쟁 | 가격 경쟁 불참, "안전의 가치"로 프리미엄 정당화 |
| 접근성 차이 | 유성구 환자에게는 오히려 접근성 우위, 주차 편의 강조 |
| SNS/리뷰 격차 | 초기 6개월 블로그 콘텐츠 집중 → 점진적 리뷰 축적 |

#### vs. 오체안 (외상/재건 직접 경쟁)

| 경쟁 이슈 | 대응 전략 |
|-----------|-----------|
| 10,000건 수술 실적 | 종합병원 인프라(입원, 마취, 응급) 안전 우위 강조 |
| 양성종양 인지도 | "조직검사+수술 원스톱" 종합병원 장점 소구 |
| 야간진료 | 종합병원 응급실 연계 24시간 외상 대응 어필 |

#### vs. 을지대병원 (포지션 경쟁)

| 경쟁 이슈 | 대응 전략 |
|-----------|-----------|
| 동일 포지션 선점 | 유성구 입지 독점성 + 빠른 예약 + 전담 전문의 직접 진료 강조 |
| 대학병원 신뢰도 | 가톨릭중앙의료원 수련 경력으로 동급 학술 배경 어필 |
| 기존 환자 베이스 | 을지대 대기시간/복잡성 불만 환자 흡수 기회 |
