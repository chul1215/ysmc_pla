## 1. 사이트맵 (Sitemap) 구성 전략

우종설 성형외과의 메뉴 구조를 유성선병원의 진료 우선순위(설문조사 기반)에 맞춰 재배치했습니다.

| 구분 | 우종설 성형외과 (벤치마킹) | **유성선병원 성형외과 (확정)** | **변경 사유 (전략)** |
|:-:|:-:|:-:|:-:|
| **1** | **ABOUT** (인사말/둘러보기) | **ABOUT** (의료진/안전시스템) | 대학병원 출신 의료진 + **종합병원 안전 시스템(응급/마취)** 강조 |
| **2** | **SURGERY** (Eye/Nose/Line/Petit) | **CENTER** (Eye/Lifting/Trauma/Skin) | 설문조사 우선순위에 따라 **'코(Nose)' 대신 '리프팅/외상'** 배치 |
| **3** | **GALLERY** (리얼셀카/전후사진) | **ON-AIR** (닥터칼럼/치료사례) | '성형모델' 모집보다는 **신뢰도 중심의 교육형 콘텐츠(영상/칼럼)** 배치 |
| **4** | **CONTACT** (상담신청) | **CONTACT** (예약/오시는길) | 종합병원 위치 안내 및 원스톱 예약 시스템 강조 |

## 3. 메인 페이지 구조 — 2카테고리 스플릿 배너 (확정)

기존 "5대 센터 나열" 방식에서 **2카테고리 스플릿 구조**로 전환. *(체리성형외과 참고)*

| 카테고리 | 포함 진료축 | 컬러 톤 | 태그라인 |
|---------|-----------|--------|---------|
| **미용·성형** | 눈 · 리프팅 · 쁘띠/스킨 | 테라코타 레드 | "아름다움을 섬세하게 설계합니다" |
| **치료·재건** | 외상 · 재건/종괴 | 다크 네이비 | "종합병원의 안전함으로 치료합니다" |

**스플릿 배너 UX 특징:**
- 데스크톱: 좌우 패널 호버 시 해당 섹션 확장(1.6배) + CTA 버튼 출현
- 모바일: 상하 2단으로 전환, 두 카테고리 동일 비중 표시

```
┌─────────────────────┬─────────────────────┐
│     미용·성형        │      치료·재건       │
│  (테라코타 레드)     │   (다크 네이비)      │
│  · Eye Center       │  · Trauma           │
│  · Lifting          │  · Reconstruction   │
│  · Petit/Skin       │                     │
│ "아름다움을 섬세하게" │ "종합병원의 안전함으로"│
└─────────────────────┴─────────────────────┘
    ↕ 호버 시 확장 (1.6배) + CTA 버튼 출현
```

## 4. 5대 진료축 전략 역할 (확정)

| # | 진료축 | 대표 키워드 | 전략적 역할 | 컬러 |
|---|--------|-----------|-----------|------|
| 1 | **쁘띠/스킨** | 보톡스, 레이저, 필러 | 반복 내원 · 유입 확대 (입구 역할) | 블루 |
| 2 | **리프팅** | 울쎄라, 써마지 | 고수익 · 프리미엄 신뢰 형성 | 퍼플 |
| 3 | **눈** | 쌍꺼풀, 하안검 | 시그니처 전환축 (핵심 상담 유도) | 앰버 |
| 4 | **외상** | 안면미세봉합, 안면골절 수술 | 종합병원만의 독보 차별화 | 레드 |
| 5 | **재건/종괴** | 몽우리, 종괴 | 의료 신뢰 정점 · 불안 해소 | 에메랄드 |

## 6. 히어로 섹션 (확정)

- **카피**: "대학병원의 안전함에 섬세함을 더하다"
- **서브**: 가톨릭중앙의료원 출신 전문의가 상담부터 수술, 회복까지 직접 케어합니다.
- **이미지**: 신뢰감 있는 의료진의 상반신 프로필 또는 첨단 수술실 배경

## 7. 각 상세 페이지 공통 구조 (확정)

```
1. 고정 헤더 / GNB 네비게이션
2. 히어로 섹션 (핵심 태그라인 + 핵심 메시지)
3. 수술 원칙 섹션 (3가지 기준 카드)
4. 특장점 카드 섹션
5. 수술 유형 그리드
6. FAQ 아코디언
7. CTA 섹션 (상담 예약 · 전화 · 카카오)
8. 푸터
```
